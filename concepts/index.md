# Concepts

* [Async/Await](async_await.md) - Syntactic sugar for writing Promise-based asynchronous code with suspend and resume semantics.
* [Closures](closures.md) - Functions that retain access to variables from their enclosing lexical scope after the outer function returns.
* [Coercion](coercion.md) - How JavaScript converts values between types implicitly and explicitly at runtime.
* [CommonJS](commonjs.md) - The require/module.exports module system used in Node.js, compared conceptually with ES modules.
* [Data Types](data_types.md) - Primitive and object types in JavaScript, including typeof behavior and value vs. reference semantics.
* [Event Loop](event_loop.md) - How JavaScript schedules asynchronous work on a single thread using the call stack, task queues, and microtasks.
* [Execution Context](execution_context.md) - The runtime frame for executing code, including the call stack, environment records, and this binding.
* [Functions](functions.md) - Function declarations, expressions, arrow functions, and how callable objects participate in execution.
* [Garbage Collection](garbage_collection.md) - How JavaScript runtimes reclaim heap memory when objects are no longer reachable.
* [Hoisting](hoisting.md) - How declarations are registered in scope during the creation phase before code execution runs.
* [Import and Export](import_export.md) - Static export declarations, import forms, re-exports, and live binding semantics in ES modules.
* [Inheritance Model](inheritance_model.md) - Prototypal inheritance, class syntax, extends, and how behavior is shared in JavaScript.
* [Memory Model Basics](memory_model.md) - Stack vs. heap storage, references, bindings, and how values live in the JavaScript runtime.
* [ES Modules](modules.md) - ECMAScript modules with static structure, strict mode, and isolated module scope.
* [Objects](objects.md) - Plain objects, property descriptors, references, and how key-value collections behave at runtime.
* [Performance Considerations](performance.md) - Main-thread constraints, allocation patterns, async scheduling, and practical JavaScript performance habits.
* [Promises](promises.md) - Deferred single-result values with settled states and microtask-scheduled reactions.
* [Prototypes](prototypes.md) - The prototype chain, [[Prototype]] links, and how JavaScript delegates property lookup.
* [Runtime Module Evaluation](runtime_evaluation.md) - How the JavaScript engine loads, links, and evaluates modules in dependency order.
* [Scope](scope.md) - Lexical scope, block scope, and how the engine resolves identifiers through nested environments.
* [This Binding](this_binding.md) - How the runtime determines the value of this based on call site, not definition site.
* [Variables](variables.md) - Declaring and binding names to values using let, const, and var in modern JavaScript.
